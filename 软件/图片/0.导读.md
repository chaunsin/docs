# 导读

在图像处理场景中，想要提取图片得信息（比如获取图片的拍摄时间、经纬度、图片大小等等），就需要了解图片中得元数据存储方式，以及相关的编码方式。
这是程序员要必要掌握的基础知识。

目前主流的存储图线元数据的规范方式有：

- IPTC
- EXIF
- TIFF
- XMP
- ICC

另外还有一些特殊的规范方式：

- Makenote
- Photoshop 3.0
- 其他

# 工具

## 1. `exiftool`

是一个用于查看、编辑、删除、复制、移动、重命名、批量处理图片、视频、音频、文档等等文件的工具。

官方地址：https://exiftool.org/

## 2. `exiv2`

和 `exiftool` 类似，`exiv2` 也是一个用于查看、编辑、删除、复制、移动、重命名、批量处理图片、视频、音频、文档等等文件的工具。

官方地址：https://exiv2.org/

## 3. https://exifinfo.org/

一个免费的用于查看图片元数据的在线网站。支持上传图片和查看图片链接两种方式。

## 图片资源库

不管是开发者，还是测试用户，以及其他目的，如果想寻找带有不同类型的元数据文件可以访问网址进行下载

1. exiftool提供 https://exiftool.org/sample_images.html
2. Pixls提供的原始图像  https://raw.pixls.us/
3. RAW-Samples提供的原始图像,里面包含了各种相机的图片  https://www.rawsamples.ch/index.php/en/
