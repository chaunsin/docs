# 导读

从传统编写语言代码到如今的AI编写代码经历了以下阶段

1. 纯人工coding，没有AI辅助,凭借个人开发经验+先人知识沉淀、结合博文等进行开发。
2. AI辅助编程阶段，也是"viber code" 阶段，通过AI辅助进行编程。
3. 规范驱动开发阶段，通过规范驱动开发，将AI辅助编程转化为规范驱动开发。
