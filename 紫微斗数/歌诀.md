# 歌诀

列出紫薇斗术中常用歌诀。

## 五虎遁决

**用途：**

根据年干确定该年各个月份（从寅月开始）的天干。

- 紫薇斗术
- 八字

**起点：** 寅月（正月，象征“虎”）。

**核心口诀：**

- **甲己之年丙作首，**
- **乙庚之岁戊为头，**
- **丙辛必定寻庚起，**
- **丁壬壬位顺行流，**
- **若问戊癸何方发，**
- **甲寅之上好追求。**

**含义（年干 -> 该年正月天干）：**

- 年干是 甲 或 己 -> 正月是 丙寅
- 年干是 乙 或 庚 -> 正月是 戊寅
- 年干是 丙 或 辛 -> 正月是 庚寅
- 年干是 丁 或 壬 -> 正月是 壬寅
- 年干是 戊 或 癸 -> 正月是 甲寅

**快速记忆法（基于天干五合）：取合化后五行所对应的“生我之阳干”。**

- 甲己合化土： 生土之阳干为 丙火 -> 口诀“丙作首”
- 乙庚合化金： 生金之阳干为 戊土 -> 口诀“戊为头”
- 丙辛合化水： 生水之阳干为 庚金 -> 口诀“寻庚起”
- 丁壬合化木： 生木之阳干为 壬水 -> 口诀“壬位顺行流”
- 戊癸合化火： 生火之阳干为 甲木 -> 口诀“甲寅之上好追求”

**总结记忆关键：**

五虎遁 (月柱) 看年干，依据天干五合，取合化五行之 “生我阳干” 作为正月（寅月）的天干。

## 五鼠遁决

**用途：**

根据日干确定该日各个时辰（从子时开始）的天干。

- 八字

**起点：** 子时（象征“鼠”）。

**核心口诀：**

- **甲己还加甲，**
- **乙庚丙作初，**
- **丙辛从戊起，**
- **丁壬庚子居，**
- **戊癸何方发，**
- **壬子是真途。**

**含义（日干 -> 该日子时天干）：**

- 日干是 甲 或 己 -> 子时是 甲子
- 日干是 乙 或 庚 -> 子时是 丙子
- 日干是 丙 或 辛 -> 子时是 戊子
- 日干是 丁 或 壬 -> 子时是 庚子
- 日干是 戊 或 癸 -> 子时是 壬子

**快速记忆法（基于天干五合）：取合化后五行所对应的“克我之阳干”。**

- 甲己合化土： 克土之阳干为 甲木 -> 口诀“还加甲”
- 乙庚合化金： 克金之阳干为 丙火 -> 口诀“丙作初”
- 丙辛合化水： 克水之阳干为 戊土 -> 口诀“从戊起”
- 丁壬合化木： 克木之阳干为 庚金 -> 口诀“庚子居”
- 戊癸合化火： 克火之阳干为 壬水 -> 口诀“壬子是真途”

**总结记忆关键：**

五鼠遁 (时柱)看日干，依据天干五合，取合化五行之 “克我阳干” 作为子时的天干。
